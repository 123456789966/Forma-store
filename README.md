#  Forma Store — Full Stack E-Commerce

A production-ready e-commerce web application built with the MERN stack.

##  Features
- 🔐 User Authentication (JWT)
- 🛒 Shopping Cart with Coupon Codes
- 💳 Stripe Payment Integration
- 📦 Order History & Tracking
- 🔍 Search, Filters & Sorting
- 📊 Admin Dashboard with Analytics
- 📱 Fully Responsive Design

##  Tech Stack
| Frontend | Backend | Database | Payment |
|----------|---------|----------|---------|
| React 18 | Node.js | MongoDB  | Stripe  |
| Vite     | Express | Mongoose | JWT Auth|
| Axios    | REST API| Atlas    |         |
